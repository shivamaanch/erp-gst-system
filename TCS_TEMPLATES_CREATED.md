## 📞 How to Use

### **TDS Workflow:**
1. Navigate to **GST & Tax → TDS**
2. View purchase bills with TDS
3. Click "Calculate TDS" for any bill
4. Select appropriate TDS section
5. Generate TDS certificate
6. Export to Excel for filing

### **TCS Workflow:**
1. Navigate to **GST & Tax → TCS**
2. View sales bills with TCS
3. Click "Calculate TCS" for any bill
4. Select appropriate TCS section
5. Generate TCS certificate
6. Export to Excel for filing

### **Certificate Generation:**
1. Click "View Certificate" on any bill
2. Professional certificate opens in new tab
3. Print or save as PDF
4. Issue to deductee/collector
