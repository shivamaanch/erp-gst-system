How to Import GSTR-2B

Method 1: JSON from GSTN Portal
1. Login to GSTN portal
2. Go to Returns → GSTR-2B
3. Download JSON file for the period
4. Upload the JSON file here

Method 2: Excel File
1. Create Excel with columns: GSTIN, Supplier Name, Invoice No, Invoice Date, Taxable Value, IGST, CGST, SGST
2. Upload the Excel file here
