Database Backup

Create a complete backup of your ERP database. The backup file will be downloaded to your computer and also saved in the backups folder on the server.

What will be backed up:
  • All companies and financial years
  • All transactions (Sales, Purchase, Milk, Journal)
  • All masters (Parties, Items, Accounts)
  • All settings and configurations
  • Complete database structure
Backup Details
  • Format: SQLite database file (.db)
  • Filename: erp_backup_YYYYMMDD_HHMMSS.db
  • Location: Downloaded to your computer + saved in /backups folder
  • Size: Varies based on data (typically 1-50 MB)

Best Practices
  • Regular Backups: Create backups daily or before major operations
  • Safe Storage: Store backups in multiple locations (computer, cloud, external drive)
  • Test Restores: Periodically test restoring backups to ensure they work
  • Before Updates: Always backup before running utilities or updates
  • Keep Multiple: Maintain at least 3-5 recent backups
