Renumber Vouchers

Vouchers will be numbered starting from this number
How it works:
  • Select the voucher type you want to renumber
  • Set the starting number (default is 1)
  • All vouchers of that type will be renumbered sequentially
  • Numbering is based on transaction date (oldest first)
